发表于 2018-10-10 | 分类于 c语言

指针变量

指针是一种变量，也称指针变量，它的值是内存地址。指针包含的是变量的地址，而变量又有自己的具体值。变量名是直接引用了一个值，指针名是间接引用了一个值。

为什么指针有类型？

因为指针的算术运算会暗度陈仓，如果对char指针加1，指针会指向存储器中下一个地址，那是因为char就占1个字节；如果是int指针，int通常占4个字节，如果对int指针加1，编译后的代码就会对存储器地址加4

指针变量的引用

指针变量和变量一样，都可以对数据进行操作。指针变量的操作主要是通过取地址运算符&和指针运算符*来实现的。

  • &a指的是变量a的地址
  • *ptr指的是变量ptr所指向的内存单元存放的内容，开始申明的时候，如果没有给ptr赋值，就并不知道它的值是多少。

指针与数组

指针和变量的结合就有了指针变量和变量指针，那么同样，指针与数组的结合就有了数组指针和指针数组。数组指针指的是指向数组的指针，指针数组也是一组指针变量。

  1. 指向数组元素的指针
    指针可以指向变量，当然也可以指向数组和数组元素。数组的指针，指的是数组的首地址（起始地址），数组元素的指针指向该数组元素的地址。

    数组变量好比指针，当你创建了一个数组，数组变量就可以当做指针使用，它指向数组在存储器中的起始位置。当在C语言中的函数中看到这样一行代码：

    指针也可以进行算术运算，但不是像整数那样的算术运算。当指针加上一个数或者减去一个数时，并不是简单地加上或者减去其值，而是加上或者减去该整数与指针指向对象的大小的乘积。这里假设指针的初值是4000，对于aPtr+=3，如果一个整数占用4个字节，则aPtr=4000+4×3=4012。

int Sum(int a,int b);

fun=Sum;
printf(“使用函数调用的结果：%d+%d=%d\n”,a,b,Sum(a,b));
printf(“使用函数指针调用的结果:%d+%d=%d\n”,a,b,(fun)(a,b));

struct STUDENT

int main(void)
{
struct STUDENT student1; /用struct STUDENT结构体类型定义结构体变量student1/
struct STUDENT p = NULL; /定义一个指向struct STUDENT结构体类型的指针变量p/
p = &student1; /p指向结构体变量student1的首地址, 即第一个成员的地址/
strcpy((p).name, “小明”); //(p).name等价于student1.name
(p).birthday.year = 1989;
(p).birthday.month = 3;
(p).birthday.day = 29;
(p).num = 1207041;
(p).score = 100;
printf(“name : %s\n”, (p).name); //(p).name不能写成p
printf(“birthday : %d-%d-%d\n”, (p).birthday.year, (p).birthday.month, (p).birthday.day);
printf(“num : %d\n”, (p).num);
printf(“score : %.1f\n”, (*p).score);
return 0;
}
