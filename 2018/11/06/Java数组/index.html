Java数组

发表于 2018-11-06 | 分类于 Java
  1. 声明数组
    int[] a; 声明了一个数组变量。[]表示该变量是一个数组,int 表示数组里的每一个元素都是一个整数,a 是变量名,但是，仅仅是这一句声明，不会创建数组。
  1. 创建数组

创建数组的时候，要指明数组的长度。如：new int[5]

引用概念：如果变量代表一个数组，比如a,我们把a叫做引用，与基本类型不同，int c = 5; 这叫给c赋值为5，声明一个引用 int[] a; a = new int[5]; 让a这个引用，指向数组
