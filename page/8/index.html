论如何优雅地抓取一个静态网页

发表于 2018-09-20 | 分类于 数据采集
  • 静态网页VS动态网页？

  1. 静态网页是相对于动态网页而言，是指没有后台数据库、不含程序和不可交互的网页。

  2. 静态网页相对更新起来比较麻烦，适用于一般更新较少的展示型网站。

  • 如何爬取静态网页的数据？

    以中国政府网为例，它就是典型的静态网页。

import requests 

from bs4 import BeautifulSoup

headers={"User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/67.0.3396.87 Safari/537.36"}



def web_crawler(url,headers):

    response=requests.get(url).content.decode("utf-8")

    news=BeautifulSoup(response,"lxml")

    my_new=news.find_all("a",{"target":"_blank"})

    return my_new



for i in range(10):

    url="http://sousuo.gov.cn/column/31421/{}.htm".format(i)
    res=web_crawler(url,headers)
    print("-------------正在下载第{}页------------".format(i+1))
    for item in res:
        with open(r"C:\Users\黄音旋\Desktop\new_1.txt","a",encoding="utf-8") as f:
        f.write(item.string+'\n')  
    f.close
